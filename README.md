# Tafhinul Hossain Shahim - Software Engineer Portfolio

Welcome to the source code for my personal portfolio website. This project showcases my skills, projects, and experience as a software engineer, featuring an interactive 3D robot, glassmorphism design, and dynamic animations.

## ✨ Features

-   **Interactive 3D Hero:** A floating robot created with **Three.js** that follows the mouse and performs animations on click.
-   **Glassmorphism UI:** A modern, translucent interface with support for both light and dark modes.
-   **Dynamic Particle Background:** An animated particle system that interacts with mouse movements and clicks.
-   **Animated Terminal:** A "whoami" style terminal in the 'About Me' section with a dynamic typing effect.
-   **Project Showcase:** A grid-based project display with category filtering (`All`, `Web`, `AI`) and detailed modal popups.
-   **Smooth Animations:** Scroll-triggered fade-in animations for cards, timeline elements, and other sections.
-   **Fully Responsive:** Optimized for a seamless experience on desktop, tablet, and mobile devices.
-   **Live Contact Form:** Integrated with **EmailJS** for direct messaging from the website.

---

## 🛠️ Tech Stack

-   **Frontend:** HTML5, CSS3, JavaScript (ES6+)
-   **3D Graphics:** Three.js
-   **Email Service:** EmailJS
-   **Icons:** Font Awesome
-   **Version Control:** Git & GitHub

##  Running Locally

To run this project on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/tafhinul/portfolio.git
    ```
2.  **Navigate to the project directory and open `index.html` in your browser.** You can do this by double-clicking the file.
